Приложение на React + Firebase

  • Фронтенд на React

    Функционал приложения включает в себя:

    • прослушивание аудиофайлов с возможностью изменения вариаций музыкального произведения
    • режим слушателя и режим автора
    • определение схем вариаций для аудиофайлов
    • интерфейс переключение вариаций музыкального произведения в реальном времени
    • личный кабинет для загрузки своих произведений и их вариаций
