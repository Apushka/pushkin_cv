Fullstack-проект
для косметического бренда

  • Реализовал бэкенд на NodeJS, Express

    Сервер обеспечивает:

    • взаимодействие с базой данный Firebase
    • авторизацию пользователя в личном кабинете
    • создание карточки товара, наполнение карточки контентом (видео, фото, текст)
    • генерацию уникального QR-кода для каждого товара

  • Реализовал фронтенд на React

    Клиент позволяет:

    • переходить на карточку товара после сканирование QR-кода (размещается на стендах компании)
    • просматривать карточку товара без перезагрузок
    • осуществить вход в личный кабинет (для администратора) для создания, редактирования или удаление товара из базы данных
    • вести статистику с помощью Яндекс.Метрики

  • Деплой проекта на сайт компании
